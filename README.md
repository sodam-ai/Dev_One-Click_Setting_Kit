# Dev_One-Click_Setting_Kit — 바이브코딩 환경 원클릭 설정 키트

> Windows PC에서 AI로 코딩(“바이브코딩”)을 시작하는 데 필요한 개발 도구들을 **파일 하나 더블클릭**으로 자동 설치해 주는 도구입니다.

## 1. 이게 뭔가요?

“바이브코딩”은 **AI에게 한국어로 말하듯 요청해서 프로그램을 만드는 방식**입니다.
그런데 AI 코딩을 하려면 먼저 PC에 여러 **개발 도구**(Git, Python, Node.js 등)를 깔아야 합니다.
하나씩 찾아 설치하면 처음엔 30분~1시간이 걸리고 실수도 잦습니다.

이 키트는 **`dev-one-click-setting-kit.bat` 파일 하나**만 실행하면:

- 필요한 도구를 **자동으로 찾아 설치**하고,
- 이미 있는 도구는 **건너뛰거나 최신으로 올려**주고,
- 설치가 끝나면 **결과 요약 파일**을 만들어 줍니다.

> ⚠️ **Windows 전용**입니다. macOS·Linux에서는 작동하지 않습니다.

**용어 풀이 (처음이면 이것만 알면 됩니다)**

| 용어 | 아주 쉽게 |
|------|----------|
| 터미널 / 콘솔 | 명령을 글자로 입력하는 검은 화면 |
| winget | Windows에 기본 내장된 “프로그램 자동 설치기” |
| CLI | 마우스 대신 글자 명령으로 쓰는 프로그램 |
| LTS | “오래 지원되는 안정 버전”(장기 안전판) |

## 2. 사전 준비물

| 항목 | 필요 조건 |
|------|-----------|
| 운영체제 | **Windows 10 (21H2, 빌드 19044 이상)** 또는 **Windows 11** |
| 인터넷 | **필수** (winget이 인터넷에서 도구를 내려받습니다) |
| 디스크 여유 공간 | 왕초보 3GB / 전체 설치 시 약 7GB 이상 권장 |
| 권한 | **관리자 권한 불필요**(대부분). 일부 도구만 실패 시 관리자 실행 |
| winget | Windows 10/11에 보통 내장. 없으면 [설치 안내](#14-문제--오류-대처-방법) 참고 |

> Windows 10 오래된 버전(21H1 이하)은 winget이 불안정합니다. 이 경우 키트가 경고하고 계속 여부를 물어봅니다.

## 3. 필요 프로그램(설치되는 도구 목록)

설치는 **4단계 레벨(누적)**로 나뉩니다. 상위 레벨은 하위 레벨을 모두 포함합니다.

| 레벨 | 누적 개수 | 추가되는 도구 | 예상 시간 |
|------|:--------:|--------------|:--------:|
| **왕초보** | 5 | Git · Python 3 · Node.js LTS · VS Code · Windows Terminal | ~7분 |
| **중급** | 11 | + Git LFS · GitHub CLI · PowerShell 7 · pnpm · Bun · Ollama | ~15분 |
| **고급** | 16 | + Java 21 LTS · Go · Rust · Flutter+Dart · Stripe CLI | ~35분 |
| **올인원** | 18 | + Ruby · PHP | ~45분 |

**추가로 고를 수 있는 것 ([5] 선택 설치)**

- winget 도구: **uv**(Python 패키지 관리)
- npm 도구(Y/N 선택): Vercel CLI · Supabase CLI · Stripe SDK · Resend SDK · Railway CLI · Clerk · Prisma · Uploadthing

**특정 버전 설치 ([V] 버전 선택 설치)** — 안정/LTS 버전만

- **Python**: 3.11 / 3.12 / 3.13
- **Java(JDK)**: 8 / 11 / 17 / 21 (전부 LTS)
- **Ruby**: 3.2 / 3.3

**설치 후 자동으로 되는 것 (Post-install)**

- Git 줄바꿈 설정(`core.autocrlf true`) 자동
- Git 이름·이메일은 **자동 저장하지 않고 안내만** (개인정보 보호)
- Python `pip` 최신화, Node의 광고 메시지 끄기(`npm config set fund false`)
- **Claude Code CLI 자동 설치** (성공/실패를 정직하게 표시)
- Ollama 로컬 AI 모델은 **안내만**(자동 다운로드 안 함 — 용량 큼)
- 설치된 도구가 잘 인식되는지 **PATH 검증** 출력

> AI 에디터(Cursor, Claude Desktop 등)는 **프로그램**이라 자동 설치하지 않고 **[8] 직접 다운로드**에서 공식 페이지를 열어 줍니다.

## 4. 다운로드 방법

**방법 A — Releases(권장)**
저장소 페이지 오른쪽 **Releases** → 최신 버전 → **Assets**의 `dev-one-click-setting-kit.bat` 클릭해 저장.

**방법 B — ZIP**
`< > Code` 버튼 → **Download ZIP** → 압축 해제 → 폴더 안의 `dev-one-click-setting-kit.bat` 사용.

> 파일 하나(`.bat`)만 있으면 됩니다. 다른 파일은 설명서·개발 기록입니다.

## 5. 빠른 시작 (3분 요약)

1. `dev-one-click-setting-kit.bat`를 **더블클릭**.
2. 파란 경고창(SmartScreen)이 뜨면 **[추가 정보] → [실행]**.
3. 메뉴가 나오면 처음이면 **`A`**(가장 쉬운 추천 설치) 또는 **`1`**(왕초보) 입력 후 **Enter**.
4. **`Y`**로 설치 시작 → 이미 설치된 것 처리엔 그냥 **Enter**(건너뜀).
5. 끝나면 안내대로 **새 터미널**을 열어 사용 시작.

---

## 6. 실행 방법

1. **파일 받기** → [4. 다운로드](#4-다운로드-방법).
2. **더블클릭 실행.** 관리자 권한은 대부분 불필요. 일부 실패 시 파일 **우클릭 → “관리자 권한으로 실행”**.
3. **SmartScreen 파란 창**이 떠도 정상입니다: **[추가 정보] → [실행]**.
4. 검은 화면(터미널)에 메뉴가 뜹니다. 원하는 번호/문자를 입력하고 **Enter**.

> 인터넷으로 내려받기 때문에 시간이 걸릴 수 있습니다. **창을 닫지 마세요.**

## 7. 사용 방법 (메뉴 설명)

실행하면 나오는 메인 메뉴:

```
[1] 왕초보 설치     처음 시작 (5개,  ~7분)
[2] 중급 설치       (11개, ~15분)
[3] 고급 설치       (16개, ~35분)
[4] 올인원 설치     (18개, ~45분)
[5] 선택 설치       원하는 번호만 골라서
[6] 업데이트        설치된 도구를 안정(LTS) 버전으로
[7] 제거            도구 삭제 (개별/전체)
[8] 직접 다운로드   공식 사이트 URL 목록 (winget 불가 시)
[9] 설치 확인       O/X + 버전 표시
[0] 종료
[A] 가장 쉬운 추천 설치   처음이면 이거! (기본 5종 + AI 안내)
[V] 버전 선택 설치        특정 안정버전 (Python/Java/Ruby)
```

| 메뉴 | 설명 |
|------|------|
| **[1]~[4]** | 레벨별 일괄 설치. 설치 전 목록·소요시간을 보여주고 **Y/N** 확인 |
| **[A]** | 기본 5종 설치 + Cursor·Claude Desktop 다운로드 페이지 열기 |
| **[5]** | 번호를 쉼표로 입력(예: `1,3,8`). **Enter만 누르면 권장 기본팩(1~5)**. 고른 도구 **이름을 미리 보여준 뒤** 설치 |
| **[V]** | Python/Java/Ruby의 **특정 안정 버전** 선택. 잘 모르면 그냥 Enter(권장값) |
| **[6]** | 이 키트가 설치한 도구만 **안정(LTS) 버전으로** 업데이트 |
| **[7]** | 개별/전체 제거 |
| **[8]** | winget으로 안 될 때 공식 다운로드 페이지를 브라우저로 엶 (URL은 **Ctrl+클릭**) |
| **[9]** | 각 도구 설치 여부(O/X)와 버전 확인 |

## 8. 작동 방법 & 명령어

이 키트는 **메뉴 번호만 입력**하면 되므로 외울 명령어가 없습니다.
설치가 끝난 뒤, **새 터미널**에서 아래로 잘 깔렸는지 확인할 수 있습니다.

```
git --version
python --version
node --version
claude --version
```

**Git 첫 사용 시(1회만)** — 이름·이메일 설정(키트는 안내만 하고 자동 저장하지 않음):

```
git config --global user.name  "홍길동"
git config --global user.email "your@email.com"
```

## 12. 아키텍처

- **단일 파일**: 순수 Windows 배치 스크립트(`.bat`) 1개. 별도 설치/빌드 없음.
- **패키지 관리**: **winget**(Microsoft 공식·서명 검증) 사용. 각 도구는 공식 소스에서 설치.
- **구조(서브루틴)**: 메뉴 → `:PRE_CHECK`(사전 점검) → `:INSTALL`/`:NPM_INSTALL`(설치·재시도) → `:POST_*`(설치 후 설정) → `:MAKE_REPORTS`(리포트) → `:PATH_CHECK`(검증).
- **이식성**: `%~dp0` 기반이라 파일 위치·이름을 바꿔도 작동.
- **인코딩**: CP949(한국어 Windows 기준). 영어 Windows에서는 메뉴 글자가 깨질 수 있음(알려진 제약 — [14 참고](#14-문제--오류-대처-방법)).

## 13. 보안 / 데이터 흐름

- **관리자 권한 불필요**(사용자 범위 설치). 기업/학교 PC 대응.
- **공식 소스만**: 모든 설치는 `--source winget`(Microsoft 서명 검증). 임의 URL 실행 경로 없음.
- **입력 최소화**: 사용자는 **메뉴 번호만** 입력. 임의 명령 실행 통로 없음.
- **개인정보 보호**: Git 이름/이메일은 **자동 저장하지 않고 안내만**.
- **외부 전송 없음**: 만들어지는 파일은 **로컬**(리포트/로그)뿐. 서버로 보내지 않음.
- **리포트 내용**: PC 이름(COMPUTERNAME)·설치 결과만. **사용자명·홈 경로·비밀번호 등 민감정보 미포함.**
- **시크릿 없음**: 코드/문서에 API 키·토큰·비밀번호가 들어있지 않습니다.

## 14. 문제 / 오류 대처 방법

| 증상 | 원인(가능성) | 대처 |
|------|-------------|------|
| 파란 경고창(SmartScreen) | Windows 보호 기능 | **[추가 정보] → [실행]** (정상) |
| `winget을 찾을 수 없습니다` | 앱 설치 관리자 없음 | Microsoft Store에서 **“앱 설치 관리자”** 설치, 또는 https://aka.ms/getwinget |
| 설치 중 창이 갑자기 닫힘 | 오류 발생 | 같은 폴더의 `install-log-날짜.txt` **마지막 줄** 확인 |
| 특정 도구만 설치 실패 | 네트워크·정책·winget 일시 오류 | 자동 1회 재시도됨. 그래도 안 되면 **[8] 직접 다운로드** |
| 관리자여도 설치 안 됨 | 백신이 `.bat` 차단 | 백신 허용 목록 추가 또는 잠시 실시간 보호 해제 후 재시도 |
| **메뉴 글자가 깨져 보임** | **영어(비한국어) Windows** | 한국어 Windows에서는 정상. 영어 Windows는 알려진 제약(향후 PowerShell 런처로 개선 예정) |
| 디스크 공간 부족 경고 | 여유 공간 미달 | 공간 확보 후 재실행(레벨별 3/4/6/7GB 기준) |

> 오류 신고 시: 스크린샷 대신 `install-log-날짜.txt`를 첨부하면 원인 파악이 쉽습니다. (민감정보 없음)

## 15. 자주 묻는 질문 (FAQ)

**Q. 코딩을 한 번도 안 해봤는데 써도 되나요?**
네. **[A]** 또는 **[1]**만 눌러도 기본 환경이 갖춰집니다.

**Q. 다시 실행해도 되나요?**
네. 이미 설치된 것은 기본값 **건너뜀**이라 안전합니다.

**Q. 관리자 권한이 꼭 필요한가요?**
대부분 아니요. 일부 도구만 실패할 때 관리자 실행을 안내합니다.

**Q. 특정 버전(예: Python 3.11)을 깔 수 있나요?**
네. **[V] 버전 선택 설치**에서 안정 버전만 고를 수 있습니다.

**Q. 설치한 걸 지우려면?**
**[7] 제거** → 개별 또는 전체.

**Q. 인터넷이 꼭 필요한가요?**
네. winget이 인터넷에서 도구를 내려받습니다.

**Q. macOS/Linux에서 되나요?**
아니요. **Windows 전용**입니다.
